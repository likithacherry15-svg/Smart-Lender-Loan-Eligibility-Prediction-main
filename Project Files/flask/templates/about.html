About Smart Lender

Smart Lender is a Machine Learning based web application developed to predict whether an applicant is eligible for a loan.

Technologies Used

  • Python
  • Flask
  • Scikit-Learn
  • Random Forest
  • Pandas
  • NumPy
  • Bootstrap 5

Machine Learning Models

  • Decision Tree
  • Random Forest ⭐
  • K-Nearest Neighbors
  • XGBoost

Best Model

Random Forest achieved the highest testing accuracy of 77.24% and was selected as the final deployment model.
